Consensus fix order

  • 01 Verify ffmpeg installed: sudo apt install ffmpeg — 2 minutes, prevents silent crash on first motion event
  • 02 Fix uv venv to see picamera2: uv venv --system-site-packages && uv sync
  • 03 Fix the cooldown/recording bug — split detect() from new_event_allowed() — core product correctness
  • 04 Fix camera.close() to call stop() then close()
  • 05 Anchor CLIPS_DIR to project root; fix .gitignore; reconcile code vs. docs (clips/ vs. 00-clips/)
  • 06 Add CI workflow — same pattern as the tutorial, skip camera tests
  • 07 Fix email attachment filename (os.path.basename), wire up config.FPS, clean up ruff findings
  • 08 Supervised manual test on hardware — confirm clip duration matches real motion, email arrives
