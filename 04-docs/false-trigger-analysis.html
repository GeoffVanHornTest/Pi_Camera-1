Key Findings

Scoring system failed — 17 of 18 confirmed false triggers misclassified. Highest score in entire session was 5 (one clip). Nothing reached FALSE_TRIGGER. The suite needs re-calibration before it can be relied on.
in_cluster fires on 41 of 42 clips. The person was active for hours, so every clip is in a cluster. Signal cannot distinguish "headlights passing repeatedly" from "person active all morning." Currently adds +1 to everything.
wall_zone_pct > 60% never fired in the entire session. All 42 clips classified MIXED (31–55% wall). Threshold is miscalibrated for this room geometry. Needs to drop to ~42%.
Type B: MOG2 background drift detected motion in an empty room for up to 73% of a clip. Gradual morning light rise causes the unadapted background model to register whole-frame luminance change as movement. These clips score 1–2 (PERSON) and look identical to real person clips.
Reflection heuristic fired on 7 confirmed false triggers but only added +1 each. Also fired on real person clips — not a clean discriminator on its own. Needs to be gated on gap_ratio to avoid false positives.
gap_ratio signal is fundamentally sound. Correctly identified 8 empty-room clips. Problem is these max out at score 4 because other signals don't compound. Threshold and combination logic needs adjustment, not the metric itself.

Scoring Fixes (F1–F4)

Fix F1
Combine gap + SHORT

gap > 0.85 AND SHORT → +3 instead of +2 and +1 independently. A short clip with a mostly-empty room is a much stronger joint signal.

Fix F2
Lower reflection gate

Change reflection gate from gap > 0.50 to gap > 0.35. Type B clips have gap 0.27–0.44 and reflection 86–98% — the heuristic fires but the gate blocks it.

Fix F3
Lower wall zone threshold

Change wall_zone_pct > 60% to > 42%. Zero clips crossed 60% in 18 confirmed false triggers. At 42%, four clips would have scored +1.

Fix F4
MOG2 learning rate (root cause)

Try bg.apply(frame, learningRate=0.02) in motion_detector.py. Default auto rate adapts too slowly to gradual morning light change. Addresses Type B at the source.
